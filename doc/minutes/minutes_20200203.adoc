=== Measurement
There should be metrics implemented in order to measure the pipeline performance end to end.
A performance measurement to a point of reference would be interesting.
The dataset for this measurement does not matter. What counts is that one gets a point of reference.
E.g.: How many pictures have been detected correctly?

Possible questions for the defence:

** How many pictures have been detected correctly?
** How many pictures have been classified correctly?
** How was the training conducted systematically?
** How was the validation conducted systematically?
** Which parameters have been set?
** Loss function J: Which regularisation k has been taken?
** Why is the chosen metric a good choice?
